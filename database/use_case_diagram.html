Use Case Descriptions

User Management

Manage User Accounts: Admin can create, update, and deactivate user accounts.

Update Profile: All users can update their own profile information.

Student Management

Register Student: Admin/Teacher can register new students in the system.

Update Student Records: Admin/Teacher can update student information.

View Student Progress: Both Admin/Teacher and Parents can view student progress reports.

Progress Reports

Create/Edit/Delete Progress Reports: Admin/Teacher can create, edit, and delete progress reports.

View Progress Reports: Both Admin/Teacher and Parents can view progress reports.

Attendance

Mark Attendance: Admin/Teacher can record daily attendance.

View Attendance Records: Both Admin/Teacher and Parents can view attendance history.

Events

Create/Manage Events: Admin/Teacher can create and manage events.

View Events: All users can view upcoming events.

Register for Events: Parents can register their children for events.

System

Manage System Settings: Admin can configure system-wide settings.

View Dashboard: All users can view their personalized dashboard.
